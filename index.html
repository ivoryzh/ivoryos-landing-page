Our Vision

The standard software ecosystem for self-driving labs

IvoryOS is the first and only truly adoptable user interface for the self-driving lab (SDL) community. Our goal is to standardize the user experience across labs, instruments, and optimizers. We want to make it easy for anyone to build, run, and optimize workflows with flexibility but zero code. With growing community effort toward building compatible drivers, more standard optimizer I/O, we envision IvoryOS evolving into the standard software ecosystem for SDL builders and scientists.

In this future, a true plug-and-play:
