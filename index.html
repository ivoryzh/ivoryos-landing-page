About IvoryOS

IvoryOS addresses the fundamental challenge of creating flexible user interfaces for self-driving laboratories.

Our solution dynamically displays functions and parameters for initialized Python modules, requiring minimal modification to existing workflows.

  • Dynamic function discovery and UI generation
  • Visual workflow design with drag-and-drop interface
  • Real-time execution monitoring and control
  • AI-powered code and campaign generation

Hein Lab at the University of British Columbia is at the forefront of automated chemistry and self-driving laboratory development. Our research group focuses on developing innovative automation solutions that accelerate chemical discovery and optimize synthetic processes.
